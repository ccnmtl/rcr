>> System and Browser Requirements <<

System requirements

For Windows XP or later.
64 MB of RAM (128 or higher recommended)
Monitor: 800x600 pixel resolution recommended
Browser: Internet Explorer or Mozilla Firefox 3
Plug-ins: RealOne Player, Adobe Acrobat Reader 7 or later

For Macintosh
Mac OS X or higher
64 MB of RAM (128 or higher recommended)
Monitor: 800x600 pixels resolution recommended
Browser: Apple Safari 4 or Mozilla Firefox 3
Plug-ins: RealOne Player, Adobe Acrobat Reader 7 or later


Using Firefox
The Firefox browser is free. Download the latest version of Firefox and follow the simple download instructions.



Using Internet Explorer
The Internet Explorer browser is free. Download Internet Explorer and follow the simple download instructions. PC users, please be aware that if you have a previous version of Internet Explorer, it will be overwritten when you download a later version.

Broken link or site not found
While we make an effort to keep links to external sites up-to-date, they may change without our knowledge—or be temporarily offline. If you encounter a problem linking to an external site, please e-mail ccnmtl-rcr@columbia.edu with the name of the MSE and a description of the page with the broken link.
